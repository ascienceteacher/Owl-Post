# Owl Post

Helping students build stronger connections between school and home through meaningful communication.

This GitHub Pages package displays the complete interactive Owl Post Google Apps Script web app. It does not replace the app with a static mockup, so the original student prompts, login flow, preview, submission process, and teacher approval hub remain available.

## Upload to GitHub

Upload these files to the root of the `Owl-Post` repository:

- `index.html`
- `owl-post-rowlie.png`
- `.nojekyll`

GitHub Pages address:

`https://ascienceteacher.github.io/Owl-Post/`

The connected Apps Script deployment is already included in `index.html`.
