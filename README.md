# Owl Post

![Owl Post](owl-post-rowlie.png)

**Helping students build stronger connections between school and home through meaningful communication.**

Owl Post gives students a guided way to write meaningful updates home. Teachers review each message before it is sent to a parent or guardian.

## Design

- Rowlie is shown without goggles.
- No science-specific wording appears on the landing page.
- Luckiest Guy headings and a Mondrian-inspired layout are used throughout.
- The page is responsive for iPads, laptops, and phones.

## Publish with GitHub Pages

Upload these files to the root of the repository:

- `index.html`
- `owl-post-rowlie.png`
- `README.md`
- `.nojekyll`

Then publish from **Settings → Pages → Deploy from a branch → main → /(root)**.

The Apps Script web-app address is already connected inside `index.html`.
